Technical Writing Portfolio – Lars Nilsson

A collection of documentation and tooling projects: DITA-based manuals, an S1000D DMC generator, and interactive courses on ILS and S1000D fundamentals.

From IT support to technical writing

I’ve spent years in IT support, troubleshooting systems, working with identity management, and writing internal documentation in Confluence. That experience taught me what happens when documentation fails: tickets pile up, users get stuck, and critical knowledge becomes hard to find. Today, I’m building on that foundation by moving into technical writing and ILS through independent study and hands-on projects like the ones shown here.

DMC Generator — S1000D Tool

An interactive browser-based tool for building and validating S1000D Data Module Codes (DMC). Supports all DMC segments with inline validation, colour-coded output, and a DMRL (Data Module Requirements List) builder with CSV and XML export. Built as a practical learning tool for working with S1000D Issue 5.0.
