Technical Writing Portfolio – Lars Nilsson

A collection of structured documentation and tooling projects: S1000D-inspired XML tools, DMC support, DITA-based manuals, and interactive ILS learning resources.

Structured documentation, ILS thinking, and practical tooling

I’ve spent years in IT support, troubleshooting systems, working with identity management, and writing internal documentation in Confluence. That background shaped how I approach technical writing: documentation should be clear, structured, maintainable, and useful in real workflows. Today, I’m building on that foundation through DITA/XML, S1000D-inspired projects, ILS study, and small web tools that make technical information easier to inspect and understand.

Selected Projects

S1000D Web Toolkit — XML Viewer

A browser-based viewer for S1000D-style XML data modules. The tool parses local XML files and presents DMC breakdown, issue data, applicability, safety information, requirements, procedures, description content, and raw XML/JSON. Built as a static HTML/CSS/JavaScript project for GitHub Pages.

S1000D XML ILS HTML/CSS/JS GitHub Pages

DMC Generator — S1000D Companion Tool

An interactive browser-based tool for building and validating S1000D Data Module Codes (DMC). Supports all DMC segments with inline validation, colour-coded output, and a DMRL (Data Module Requirements List) builder with CSV and XML export. Works as a focused companion to the S1000D Web Toolkit.

S1000D DMC DMRL Tooling HTML/CSS/JS
View DMC Generator →

ILS Fundamentals – Self-Learning Project

An interactive self-learning course on Integrated Logistics Support (ILS) for technical writers. Eight modules covering LSA/LORA, S1000D, configuration management and IETP tools, built around a fictional Saab radar system (ARTHUR Mk II) as a running case study.

ILS S1000D LSA/LORA HTML/CSS/JS GitHub Pages
View ILS Fundamentals →

S1000D for Beginners

Interactive beginner-friendly course that explains S1000D in clear language through a simple bicycle example. Covers core concepts, data module codes (DMC), information codes, BREX, and structured XML basics.
